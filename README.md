## 📌 Project Overview

This project analyzes rental datasets to uncover patterns in customer behavior, category popularity, and store performance.

The analysis focuses on transforming raw transactional data into insights that support business decisions.

Completed while progressing through the **Udacity Data Analyst Nanodegree**.

---

## 🎯 Business Questions

This analysis explored questions such as:

- Which film categories generate the highest rental activity?
- How does rental performance vary by store?
- Which customer behaviors occur most frequently?
- How do rental durations compare across categories?
- What trends emerge from quartile and segmentation analysis?

## 📈 Key Findings

- Family-oriented categories showed stronger rental frequency, suggesting higher customer demand.

- Rental volume differed across store locations, indicating varying customer activity levels.

- Quartile analysis revealed meaningful differences in rental duration patterns.

- Aggregation techniques identified categories contributing most to overall rentals.

- SQL transformed raw datasets into actionable business insights.
